# 🛍️ Customer Shopping Behavior Analysis & Dashboard

## 📌 Project Overview

This project analyzes customer shopping behavior by transforming raw data into meaningful insights using data cleaning, database management, and visualization techniques.
It demonstrates an end-to-end data analytics workflow from preprocessing to dashboard creation.

---

## 🎯 Objectives

* Clean and preprocess raw customer data
* Handle missing values effectively
* Perform feature engineering for enhanced analysis
* Store structured data in a relational database
* Develop an interactive dashboard for insights

---

## 🛠️ Tools & Technologies

* **Python** (Pandas)
* **SQLite** (Database)
* **DBeaver** (Database Management)
* **Power BI** (Data Visualization)
* **Gamma AI** (Presentation)

---

## 🐍 Python Implementation

* Data cleaning and preprocessing using Pandas
* Missing value handling using group-wise median
* Feature engineering:

  * Age Group segmentation
  * Purchase frequency conversion (categorical → numerical)
* Column standardization and data transformation

---

## 🗄️ Database Integration

* Cleaned data stored in SQLite database
* SQL queries executed using DBeaver for analysis

---

## 📊 Power BI Dashboard

The dashboard provides interactive insights including:

* Total sales analysis
* Category-wise revenue distribution
* Customer segmentation (age groups)
* Purchase frequency trends
* Review rating analysis

---

## 💡 Key Insights

* Adult customers contribute the highest share of revenue
* Certain product categories dominate overall sales
* Majority of customers follow weekly or monthly purchase patterns
